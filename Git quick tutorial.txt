
В командной строке:

проверка версии  - "git --version"
искать помощь - "git --help"

Регистрация -  "git config --global user.email 'you@example.com'"
  		"git config --global user.name 'Your Name'"

Первое что необходимо сделать вначале рыботы с гитом это:
ВСЕ ЧЕРЕЗ КОМАНДНУЮ СТРОКУ!

Проиницилизировать репозиторий  - "git init" (необходимо находиться в нужной папке!)

Выяснить, что происходит - "git status!"  (необходимо находиться в нужной папке!)

Добавить файл, что бы git за ним началследить - "git add 'file'"

Чтобы перестал следить  - "git rm --cached 'file'"

Чтобы добавить все новые или измененные файлы - "git add ."

Совершить комит - "git commit -m 'message'"

Для игнорирования выйлов нужно в папке репозитория создать файл ".gitignore",
 и в нем перечислить все файлы которые необходимо проигнорировать. Но сам файл ".gitignore" нужно добавить.

Узнать в какой ветке я нахожусь - "git branch"

Создать новую ветку - "git branch branchName"

Удалить ветку - "git brunch -D branchName"

Переключиться на другую ветку - "git checkout branchName"

Создать новую ветку и переключиться на нее - "git checkout -b branchName"

Совместить ветки - "git merge branchName"

Добавить удаленный репозиторий - "git remote add origin git@github.com:afrojeeus/репозиторий" (Ссылка есть в самом репозитории на гитхаб)

Загрузить на удаленный репозиторий - "git push -u origin имя ветки"
